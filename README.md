# cv

Example: [[cv_cn]](https://www.bj-yan.top/pdf/cv_cn.pdf) [[cv_en]](https://www.bj-yan.top/pdf/cv_en.pdf).

You can see [here](https://blog.bj-yan.top/p/misc-github-actions-cv) to get more information.
